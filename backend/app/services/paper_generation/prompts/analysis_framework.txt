# Analysis Framework Prompt

You are an expert data analyst. Transform experimental results into compelling insights.

## Input
- Experiment Results: {{results}}
- Research Questions: {{questions}}
- Method Details: {{method}}
- Comparison Data: {{comparisons}}

## Analysis Section Structure

### 5.1 Main Results Analysis
- Key findings summary
- Performance trends
- Statistical significance
- "Surprising" results discussion

### 5.2 Ablation Analysis
- Component contributions
- Sensitivity analysis
- Optimal configurations
- Design validation

### 5.3 Comparison with Baselines
- Win/loss breakdown
- When does method win?
- When does it struggle?
- Head-to-head analysis

### 5.4 Qualitative Analysis
- Visual examples
- Case studies
- Error analysis
- Success modes

### 5.5 Efficiency Analysis (if applicable)
- Computational cost
- Memory usage
- Training time
- Inference speed

### 5.6 Limitations and Broader Impact
- Method limitations
- Failure modes
- Societal impact
- Future work

## Writing Guidelines
1. Lead with the story, not the numbers
2. Every claim needs evidence
3. Discuss both successes AND failures
4. Connect results to method design
5. Avoid "cherry-picking" - show representative examples

## Analysis Checklist
- [ ] Results answer all research questions
- [ ] Statistical significance reported
- [ ] Negative results discussed
- [ ] Results connected to method design
- [ ] Limitations honestly stated
- [ ] Broader impact considered

## Output Format
Provide the complete analysis section with:
- Clear narrative structure
- Figures/tables referenced
- Honest discussion of limitations
- Forward-looking conclusion
