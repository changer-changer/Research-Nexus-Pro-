# PointVLA_2503.07511

## 1. 研究背景

**核心**：
VLA 主要依赖 2D 视觉，空间推理受限；直接重训 3D VLA 代价高且浪费已有 2D 预训练。  
**来源**：https://arxiv.org/abs/2503.07511

## 2. 研究问题

**核心**：
如何在不破坏已训练 VLA 的前提下，将点云 3D 信息注入动作专家，提高空间推理与真实鲁棒性。  
**来源**：https://arxiv.org/abs/2503.07511

## 3. 核心创新工作

**核心**：
1. **PointVLA 注入模块**：冻结 VLM 与动作专家，仅在动作专家中插入轻量点云特征模块。  
2. **Skip‑block 分析**：选择对最终动作贡献较小的层进行注入，减少对已有表征破坏。  
3. **点云特征融合**：点云编码→投影→与动作 token 融合。  
**来源**：https://arxiv.org/abs/2503.07511

## 4. 关键实验设计

**核心**：
1. **平台**：RoboTwin 仿真与真实机器人（UR5e 双臂等）。  
2. **任务**：少样本多任务、Real‑vs‑Photo、桌高变化、长时序传送带抓取。  
3. **基线**：OpenVLA、Diffusion Policy、DexVLA、3D Diffusion Policy。  
**来源**：https://arxiv.org/abs/2503.07511

## 5. 核心结果与关键Insight

**核心**：
1. 仿真与真实任务均显著超越 2D VLA 与 3D IL 基线。  
2. 少样本任务与空间变化场景中优势明显。  
**来源**：https://arxiv.org/abs/2503.07511

## 6. 待解决问题与未来方向

**核心**：
需要进一步探索更大规模 3D 数据与更复杂动作专家上的注入策略上限。  
**来源**：https://arxiv.org/abs/2503.07511
