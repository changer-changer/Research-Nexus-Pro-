# TacDiffusion_2409.11047

## 1. 研究背景

**核心**：
高精度装配依赖触觉与力控制，但多数学习策略使用运动域动作并通过阻抗控制间接实现力控制，难以在高频控制与跨任务迁移之间取得平衡。已有高精度插入研究强调力控制的重要性，但缺乏将生成模型直接用于“力域动作”的系统框架。扩散模型在多模态建模与分布学习上表现优越，但在力/触觉为主的接触操控中尚未被充分探索。  
**来源**：https://arxiv.org/abs/2409.11047

## 2. 研究问题

**核心**：
如何让扩散策略直接生成 6D wrench（力/力矩），在只用单任务示教的条件下实现高精度插入任务的零样本迁移，并解决扩散推理频率与 1000 Hz 控制回路频率不匹配的问题。  
**来源**：https://arxiv.org/abs/2409.11047

## 3. 核心创新工作

**核心**：
1. **力域扩散策略**：直接预测 6D wrench F_df，避免在运动域间接控制力。  
2. **行为树专家模仿**：以此前 1 kHz 行为树插入技能为专家，在扩散策略中继承“原语切换”与自适应特性。  
3. **动态系统滤波器**：用二阶系统在 1000 Hz 上插值扩散输出，缓解频率错配。  
4. **模型尺寸‑速度权衡**：系统性评估 DF1‑DF4 的推理频率与精度，给出实用选型指引。  
5. **关键控制链路**：扩散策略→动态系统滤波→阻抗控制+前馈力矩，形成闭环力控通路。  
**来源**：https://arxiv.org/abs/2409.11047

## 4. 关键实验设计

**核心**：
1. **硬件平台**：Franka Emika Panda；控制端 Ubuntu 20.04 + 实时内核；训练端 RTX 3090。  
2. **任务设置与对象尺寸**：五类紧配合插入对象：  
   - 训练对象：Cuboid（35×25×60 mm，0.1 mm 间隙）。  
   - 测试对象：Key（37 mm）、Cyl‑S（直径 20 mm，长度 50 mm，0.02 mm 间隙）、Cyl‑L（直径 30 mm，长度 50 mm，0.025 mm 间隙）、Prism（50 mm 长，边长 11 mm，0.05 mm 间隙）。  
3. **观测与动作**：  
   - 观测 o ∈ R18：外力/内力/末端速度（1000 Hz）。  
   - 动作：6D 前馈 wrench F_ff。  
4. **扩散模型细节**：  
   - DDPM 前向噪声与反向去噪；使用残差网络预测噪声 ϵ。  
   - 噪声估计网络为多层线性+激活结构（N 为可变隐层宽度）。  
   - 训练超参：Epoch 1500、Batch 4096、LR 1e‑3、T=50、βτ 从 1e‑4 增至 1e‑2。  
   - 训练/验证 80/20 划分；导出 ONNX 加速推理。  
5. **动态系统滤波**：  
   - F¨_ff = α(β(F_df − F_ff) − F˙_ff)，α=0.9，β=0.3。  
6. **评测方式**：每模型×物体 50 随机初始位姿 ×2 次插入，总计 25 组实验条件。  
**来源**：https://arxiv.org/abs/2409.11047

## 5. 核心结果与关键Insight

**核心**：
1. **零样本迁移成功率**：  
   - DF3：Cuboid 98%、Key 99%、Cyl‑S 97%、Cyl‑L 96%、Prism 91%，平均 95.7%。  
   - 基线：平均 83.3%。  
2. **模型规模‑速度权衡（Table II）**：  
   - DF1 503.8 Hz / DF2 297.5 Hz / DF3 141.8 Hz / DF4 51.2 Hz。  
   - 成功率随规模增加提升，但 DF4 频率过低导致整体性能回落。  
3. **效率指标**：采用“完成率/平均时间”作为效率指标，DF3 在示教任务与新物体上均优于基线。  
4. **滤波收益**：动态系统滤波在 20 个场景中 16 个成功率提升，总体提升 9.15%，并显著平滑力/力矩曲线。  
5. **关键 Insight**：力域扩散能够继承行为树原语切换逻辑，并以更快执行时间提升鲁棒性。  
**来源**：https://arxiv.org/abs/2409.11047
