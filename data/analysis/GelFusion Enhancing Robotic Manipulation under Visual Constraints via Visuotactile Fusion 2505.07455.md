# GelFusion: Enhancing Robotic Manipulation under Visual Constraints via Visuotactile Fusion

**arXiv ID**: 2505.07455

## 1. 研究背景

**领域现状**: 模仿学习（Imitation Learning）在视觉完整的场景下表现良好，但在接触丰富（Contact-Rich）场景中，常因视觉遮挡（Occlusion）或模糊（Ambiguity）而失败。
**前人研究**:

- **视触觉传感器**: 高分辨率视触觉传感器（如GelSight）提供了丰富的接触几何和纹理信息。
- **融合挑战**: 现有的多模态融合往往只是简单拼接，容易导致信息丢失或单一模态主导（Dominance），尤其在视觉受限时无法有效利用触觉补偿。
  **应用痛点**: 如何在视觉受限（如物体被遮挡、光照不佳）的情况下，有效融合触觉反馈，增强机器人操作的鲁棒性。

## 2. 研究问题

**拟解决难题**:

1. **视觉受限**: 解决视觉在接触瞬间失效或不可靠的问题。
2. **模态融合**: 设计一种机制，让触觉在视觉失效时“站出来”，补充关键的接触状态信息。
3. **触觉表征**: 如何从高维触觉图像中提取出既包含静态几何又包含动态接触事件的特征。

## 3. 核心创新工作

**算法/框架**: 提出 **GelFusion** 框架。

1. **双通道触觉特征提取 (Dual-Channel Tactile Encoding)**:
   - **几何通道 (Geometric Channel)**: 使用ResNet-18提取单帧触觉图像的高维空间特征（表面纹理、接触几何）。
   - **动态通道 (Dynamic Channel)**: 计算连续帧之间的差分（Residual），提取触觉变化的动态特征（接触开始/结束、滑动、压力变化）。
   - **融合**: 两者互补，前者描述“是什么形态”，后者描述“发生了什么动作”。

2. **Cross-Attention Fusion (交叉注意力融合)**:
   - 使用Cross-Attention机制，以视觉特征为Query，触觉特征为Key/Value（或反之，具体文中指出的上下文信息提取），提取任务相关的上下文信息。
   - 这种机制使得网络能够根据当前状态动态调整对视觉或触觉的依赖权重。

**硬件**:

- 集成自制GelSight传感器（Self-made GelSight）到Fairino FR5机械臂的夹爪上。

## 4. 关键实验设计

**实验平台**:

- **机器人**: Fairino FR5六轴机械臂。
- **传感器**: 手腕相机（Visual） + 双指GelSight（Tactile）。
- **采集方式**: 遥操作（Teleoperation）。

**任务**: 3个接触丰富且存在视觉受限的任务。

1. **Surface Wiping (表面擦拭)**: 清理桌面上难以看见的水渍/污渍（视觉对比度低）。
2. **Precise Peg Insertion (精密插拔)**: 插入U盘或类似物体，手部会遮挡目标孔位（视觉遮挡）。
3. **Fragile Object Picking (易碎品抓取)**: 抓取薯片（Chips Pick），需要极其精细的力度控制以防通过力破碎（力控要求高）。

**对比基准**:

- **Vision-Only**: 仅视觉的Diffusion Policy。
- **Concatenation Fusion**: 简单的特征拼接融合。

## 5. 核心结果与关键Insight

**核心数据**:

- **成功率提升**: GelFusion在所有任务中均优于Vision-Only和简单融合方法。
  - **Wiping**: 能更干净地清除污渍，因为触觉能感知摩擦力变化。
  - **Insertion**: 在视觉完全被遮挡的插入阶段，依靠触觉反馈调整姿态，成功率显著提升。
  - **Picking**: 能在不压碎薯片的前提下稳定抓取（利用动态通道感知微小接触变化）。

**关键Insight**:

- **动态触觉特征至关重要**: 仅仅输入静态触觉图像是不够的，明确地提取差分特征（动态通道）能帮助网络更快地识别“接触发生”和“滑动”等关键事件。
- **触觉是视觉的补位**: 在Visual-Constrained（视觉受限）场景下，触觉不仅是辅助，更是主要感知源。Cross-Attention能有效地放大这种补位作用。

## 6. 待解决问题与未来方向

**局限性**:

- **传感器耐用性**: 自制GelSight的硅胶表面在长期摩擦（如擦拭任务）下容易磨损，影响一致性。
- **泛化性**: 目前仅在特定任务和特定物体上验证，对于未见过的物体或材质的泛化能力未深入探讨。

**未来方向**:

- **Sim-to-Real**: 目前主要依靠真机采集数据训练，未来可探索在仿真中模拟GelSight从仿真迁移到真机。
- **更复杂的接触任务**: 扩展到如线缆整理、布料操作等更复杂的柔性物体操作。
