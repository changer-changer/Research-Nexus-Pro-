# SpatialVLA_2501.15830

## 1. 研究背景

**核心**：
2D 视觉 VLA 缺乏空间对齐能力，导致跨机器人、跨视角空间推理困难。论文旨在引入显式 3D 空间表征。  
**来源**：https://arxiv.org/abs/2501.15830

## 2. 研究问题

**核心**：
如何建立跨机器人一致的 3D 观测与动作表示，使 VLA 具备空间理解与跨环境泛化。  
**来源**：https://arxiv.org/abs/2501.15830

## 3. 核心创新工作

**核心**：
1. **Ego3D Position Encoding**：在相机坐标系中将深度映射到 3D 坐标，再与 SigLIP 2D 语义融合。  
2. **Adaptive Action Grids**：将 7D 动作离散为自适应空间动作 token。  
3. **大规模预训练**：基于 1.1M 真实机器人轨迹进行预训练。  
**来源**：https://arxiv.org/abs/2501.15830

## 4. 关键实验设计

**核心**：
1. **评测范围**：24 个真实机器人任务 + 3 个仿真环境。  
2. **评测能力**：零样本、多任务、跨机器人适配、空间推理。  
3. **对比基线**：OpenVLA、TraceVLA、RT‑2‑X、Octo、Diffusion Policy。  
**来源**：https://arxiv.org/abs/2501.15830

## 5. 核心结果与关键Insight

**核心**：
1. 3D 表征显著提升跨机器人迁移与零样本泛化。  
2. 动作 token 减少推理 token 数量，提高推理速度。  
**来源**：https://arxiv.org/abs/2501.15830

## 6. 待解决问题与未来方向

**核心**：
需要进一步验证更大动作空间与更大规模真实数据下的上限。  
**来源**：https://arxiv.org/abs/2501.15830
