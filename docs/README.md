## 📝 文档规范

### 文档格式
- 使用 Markdown 格式
- 标题层级：# (主标题) → ## (章节) → ### (小节)
- 代码块使用 TypeScript 语法高亮
- 表格用于对比和列表

### 文档维护
- 每次重大更新后更新 `Last updated` 时间
- 保持文档间的链接有效
- 删除过时的内容而非注释掉
