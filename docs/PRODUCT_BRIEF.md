## 三大创新（Why Now, Why Us）

### 🚀 创新一：Agent 驱动的真实阅读

**不是 RAG，不是摘要，是真正的"深度阅读"**

- 每个 Agent 必须读完 PDF 的每一个章节
- 强制提取所有表格、图表、公式
- 所有引用必须标注原文位置 [Section 3.2]
- 不确定性明确标注，绝不编造

**技术壁垒**：
- 独创的 "6部分分析法"：背景→问题→创新→实验→洞察→未来
- 多 Agent 并行架构，突破单上下文长度限制
- 质量自检清单，防止 Agent 偷懒

---

### 🚀 创新二：标准化科研仓库

**我们定义了 AI 时代的学术调研标准**

```
research-topic/
├── 论文标题-arxiv编号.pdf          # 原始PDF
├── 论文标题-arxiv编号.md           # 深度分析报告
├── _metadata.json                   # 元数据索引
└── _survey.md                       # 自动生成的综述
```

**每一篇论文附赠一份"科研体检报告"**：
- 研究背景完整梳理
- 核心创新精准提炼
- 实验数据真实提取
- 研究Gap智能识别

**成果可积累**：
- 个人知识库持续积累
- 团队知识共享
- 研究方向智能推荐

### 🚀 创新三：全流程自动化 Pipeline

**从关键词到综述，一键完成**

```
输入："diffusion policy robot manipulation"
           ↓
┌─────────────────────────────────────────┐
│  Phase 1: 文献雷达 (5分钟)               │
│  - Arxiv 智能搜索                        │
│  - 自动去重 + 版本管理                    │
│  - PDF 自动下载                          │
└─────────────────────────────────────────┘
           ↓
┌─────────────────────────────────────────┐
│  Phase 2: Agent 集群分析 (30分钟/篇)     │
│  - 启动 N 个 SubAgent 并行              │
│  - 每篇论文深度阅读 + 6部分报告          │
└─────────────────────────────────────────┘
           ↓
┌─────────────────────────────────────────┐
│  Phase 3: 智能整合 (10分钟)              │
│  - 自动生成方法对比表格                  │
│  - 识别研究 Gap                          │
│  - 生成完整综述文档                      │
└─────────────────────────────────────────┘
           ↓
输出：结构化调研报告，可直接用于论文写作
```

**时间对比**：
- 人工调研100篇论文：2周
- 虾博 + Agent 集群：1天

## 应用场景

### 场景一：博士生开题前调研

**痛点**：需要读200篇论文，时间紧，怕遗漏
**方案**：
```bash
虾博 调研 "视觉触觉融合 robot manipulation" \
    --max 200 \
    --agents 10
```
**产出**：
- 200篇论文的深度分析报告
- 方法对比表格
- 研究Gap识别
- 可直接用于开题报告

### 场景二：论文 Related Work 写作

**痛点**：Related Work 最难写，引用容易遗漏
**方案**：
```bash
# 已有论文库
虾博 生成综述 --topic "扩散策略" \
    --papers ~/research/diffusion_policy/
```
**产出**：
- 完整的 Related Work 初稿
- 引用完整性检查
- 逻辑脉络梳理

### 场景三：课题组知识管理

**痛点**：新同学进来，领域知识传递困难
**方案**：
```bash
# 构建领域知识库
虾博 建库 --domain "视触觉融合" \
    --papers ./group_papers/ \
    --share
```
**产出**：
- 可传承的领域知识库
- 新同学快速上手
- 研究方向智能推荐
